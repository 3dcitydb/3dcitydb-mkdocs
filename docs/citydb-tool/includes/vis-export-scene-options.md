| Option                                                                | Description                                                                                                                                                                        | Default value          |
|-----------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------|
| `--grid-edge-length=<meters>`                                         | Edge length in meters of one grid cell used as the leaf of the spatial aggregation tree. When omitted, the cell edge is sized so the entire dataset fits in a single root cell.    | auto-sized to dataset  |
| `--lod-refine-radius=<pixels>`                                        | Projected bounding-sphere radius (pixels) above which a tile refines to its children. Applied uniformly to 3D Tiles (geometric error) and I3S (LOD threshold). Pass `0` to disable LOD. | `56`                |
| `--clamp-to-ground`                                                   | Place each feature on the ellipsoid surface (height 0). Useful when no terrain is loaded in the viewer.                                                                            |                        |
| `--texture-scale=<factor>`                                            | Texture resolution scale factor between `0.01` and `1.0`. Lower values reduce texture size and improve loading speed in the viewer.                                                | `1.0`                  |
| `--max-atlas-size=<pixels>`                                           | Maximum texture atlas edge length in pixels, between `1024` and `16384`. Higher values pack more textures per atlas but increase GPU memory usage in the viewer.                   | `1024`                 |
| `--atlas-overflow-mode=<mode>`                                        | Strategy when a cell's textures exceed `--max-atlas-size`: `hybrid`, `split`, `flat`.                                                                                              | `hybrid`               |
| `--atlas-fallback=<strategy>`                                         | Fallback strategy for cells the split stage did not subdivide further, and for every overflowing cell under `flat` mode: `expand` or `rescale`.                                    | `expand`               |
| `--enable-shading`                                                    | Emit per-vertex normals so surfaces render PBR / Lambertian shaded. When omitted, primitives render unlit (smaller files). Auto-enabled when I3S is exported with `--slpk`.        |                        |
| `--default-color=<#rrggbb[aa]>`                                       | Default sRGB color applied to features without a texture or `X3DMaterial`.                                                                                                         | opaque white           |
| <code>--feature-type-style=&lt;type=#rrggbb[aa]><br/>[,...]</code>    | Per-feature-type sRGB color override on the no-appearance path. The key is a qualified feature type name like `bldg:Building`.                                                     |                        |
| <code>--attributes=&lt;col:src><br/>[,&lt;col:src>...]</code>         | Declarative whitelist of columns to emit in the per-feature attribute table.                                                                                                       | export all top-level   |
